# zh-TW Fighting Game Terms

Use this reference only when the source content is clearly about fighting games, FGC play, matchup talk, frame data, or tournament analysis.

## Core Rule

- Taiwan FGC usage often keeps Japanese-derived jargon.
- Prefer established community terms such as `立回`, `確反`, `壓起`, `受身`, `當身`.
- Do not flatten fighting-game jargon into generic prose if the source speaker is clearly using genre terminology.
- If a term has no stable Taiwan rendering, keeping the English shorthand is better than inventing a new translation.

## Style Notes

- `立回`, `確反`, `壓起`, `拆投`, `指令投`, `受身`, `當身` are stronger choices than generic explanatory wording.
- When the speaker sounds like they are talking lab, matchup, or tournament jargon, preserve the dense player vocabulary.
- Use official Taiwan character and move names where they exist. If the community overwhelmingly uses the Japanese or English move name, that is acceptable too.
