# ytclip-6-convert-tc-to-sc

Convert a Traditional Chinese (zh-TW) SRT subtitle file into natural mainland Simplified Chinese (zh-CN).

This step is a two-pass workflow:
- Pass 1: run the bundled script for deterministic Traditional-to-Simplified character conversion
- Pass 2: localize Taiwan wording into mainland usage, especially slang, gaming terms, streaming terms, and platform/community phrasing

Do not stop after raw character conversion. The final output should read like natural mainland Chinese, not Taiwan Chinese written in Simplified characters.

## References

- `references/zh-cn-localization.md`: Mainland localization rules for slang, vocabulary, gaming terms, streaming terminology, and phrasing.

## Input

- An SRT subtitle file in Traditional Chinese, typically the zh-TW file produced by `ytclip-4-translate-en-to-zhtw`

## Output

- An SRT file in natural mainland Simplified Chinese

## Workflow

### Step 1: Mechanical conversion

Run the conversion script:

```bash
bun ytclip-6-convert-tc-to-sc/scripts/convert.ts \
  <input-zhtw.srt> \
  -o <output-zhcn.srt>
```

The script:
1. Reads the input SRT file
2. Preserves all SRT block numbering and timestamp lines exactly as-is
3. Converts only the subtitle text lines from Traditional to Simplified Chinese
4. Writes an intermediate Simplified Chinese SRT to the output path

### Step 2: Glossary check

Before rewriting the converted file, quickly scan it for:
- Game titles, item names, skill names, map names, and other game-specific terminology
- Streamer/platform language such as `实况`, `开台`, `聊天室`, `斗内`, `按赞`
- Proper nouns, community nicknames, brand terms, and romanized terms that should stay unchanged
- OpenCC conversions that are technically valid but wrong for mainland usage

If there are ambiguous terms or known preferred translations, confirm them with the user first. If the user says to skip, proceed with best judgment.

### Step 3: Mainland localization

Read `references/zh-cn-localization.md` and revise the converted SRT in chunks of 5-10 subtitle blocks at a time.

Apply these rules:
- Keep all block numbers and timestamp lines exactly unchanged
- Edit only subtitle text lines
- Preserve meaning, tone, and factual claims
- Convert Taiwan wording into natural mainland wording where needed
- Adapt slang and exclamations naturally for mainland usage
- Use mainland gaming community terminology for games, roles, skills, and items when it is clearly established
- Use mainland streaming/content-creator terminology for livestream, chat, donations, likes, comments, and highlights
- Do not add comma `，` or full-stop `。` at the end of each subtitle line
- Do not add speaker markers
- Keep names, URLs, code, and command lines unchanged
- Do not over-localize shared internet expressions that already sound natural in both regions

When a term has no clear mainland equivalent, prefer one of these in order:
1. The official mainland Chinese term
2. The most common mainland community term
3. The original English term

### Step 4: Save

Overwrite the intermediate file generated by the script, or write to a user-specified final zh-CN path.
